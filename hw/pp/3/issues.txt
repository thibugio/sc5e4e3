1. How to update the current program state after running a function-call through Mvalue?
   Mvalue returns a number or a boolsym, and thus is not capable of producing side-effects.
   An example scenario:
   
   var x = true;
   function f() {
     x = false;
   }
   function main() {
     f();
     if (x) return 1;
     else return 0;
   }
